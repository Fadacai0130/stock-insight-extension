Privacy Policy for Stock Insight AI

Last Updated: December 4, 2025

1. Introduction

Stock Insight AI ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our Chrome extension.

2. Information We Collect

We do NOT collect, store, or transmit any personal information.

The extension only stores the following data locally on your device:

  • Optional API Key: If you choose to configure a SiliconFlow API key for AI analysis features, it is stored locally in your browser using Chrome's storage API. This key is never transmitted to our servers (we don't have any servers).

3. Third-Party Services

To provide real-time stock data and AI analysis, the extension communicates with the following third-party services:

  • Yahoo Finance API (query1.finance.yahoo.com): Used to fetch real-time stock quotes. We only send stock symbols (e.g., "AAPL", "TSLA") to this public API. Yahoo Finance has its own privacy policy.
  • SiliconFlow API (api.siliconflow.cn): Used optionally for AI market sentiment analysis, only if you provide your own API key. We do not include any API keys in the extension.

4. Data Usage

The extension:

  • Does NOT track your browsing history
  • Does NOT collect or analyze your personal data
  • Does NOT share any data with third parties (except the API requests mentioned above)
  • Does NOT use cookies or analytics tools

5. Permissions Explanation

The extension requests the following permissions:

  • storage: To save your optional API key locally on your device.
  • host_permissions: To communicate with Yahoo Finance and SiliconFlow APIs for fetching stock data.

6. Content Scripts

The extension injects a content script on web pages you visit to enable the "Smart Ticker" feature (highlighting stock symbols). This script:

  • Only activates when you manually select text
  • Does NOT read, collect, or transmit any page content
  • Only processes the text you explicitly select

7. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the extension's listing on the Chrome Web Store.
